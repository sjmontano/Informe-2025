# Prompt: Typeset / Tipografía

Propósito: generar pautas tipográficas y snippets CSS para usar Discordia (títulos) y Lato (cuerpo), con escala fluida (`clamp()`), `font-display: swap`, y reglas de legibilidad mobile-first.

Instrucciones (prompt):
- Proporciona una guía corta y práctica (ejemplos CSS) que incluya: variables CSS de tipografía, escala modular responsiva con `clamp()`, line-height recomendado, tracking/letter-spacing para cada nivel, reglas para `text-wrap` y `hyphenation`, y ejemplos de uso en componentes (Título, Subtítulo, Lead, Body, Small).
- Incluye un snippet listo para agregar en `src/styles/typography.css` y recomendaciones de tokens a definir en `DESIGN.md`.

Salida esperada:
1) Snippet CSS completo y minimal.
2) Tabla breve con tamaños por breakpoint.
3) Recomendaciones para pruebas y verificaciones (contraste, legibilidad en móvil).

Responde en español.
